Build system
------------

uImager.sln is the main solution file for Visual Studio 2017 (MFC 
support is necessary). 

Free tool to make the Unicode NSIS installers:
- http://www.scratchpaper.com
Necessary Unicode Plugins:
- nsisFirewall from http://wiz0u.free.fr/prog/nsisFirewall
- UAC from https://nsis.sourceforge.io/mediawiki/index.php?title=UAC_plug-in&oldid=19055
- KillProcDLL from https://github.com/mazinsw/KillProcDLL

Non-free tool to make the translations:
- http://www.apptranslator.com


Release procedure
-----------------

1. Update Version Number and History
   - appconst.h
   - 2x in uImager.rc2
   - CurrentVersion.nsh
   - HistoryFreeVimager.txt, HistoryContaCam.txt and translation History.txt

2. Build release target of FreeVimager and ContaCam with Visual Studio

3. Copy FreeVimager.exe and ContaCam.exe to Translation directory,
   call Update Source in appTranslator and Build all Target Files
   
4. Run Make_All.bat in Setup folder
   
5. SVN Commit as: Final release x.x.x
